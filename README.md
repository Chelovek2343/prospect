# Prospect Admissions & Student Platform

Этот репозиторий содержит исходный код веб-сайта **Prospect Admissions**. Проект представляет собой лендинг и информационную платформу для команды менторов, помогающих с поступлением в топовые университеты, а также презентует систему управления обучением (Student Information System).

🔗 **Демо версия:** [https://chelovek2343.github.io/prospect/](https://chelovek2343.github.io/prospect/)

## 📋 Описание проекта

Сайт выполняет две основные функции:
1. **Prospect Admissions:** Информирует о услугах менторов (выпускников NYU, UBC, HKU, CityU) для помощи абитуриентам.
2. **Prospect Student:** Презентует возможности программного обеспечения для онбординга студентов, управления учебными процессами и взаимодействия между администрацией, учителями, родителями и учениками.

### Основные разделы:
* **Главная (Home):** Обзор платформы, преимущества для разных ролей (Admin, Teacher, Parent, Student).
* **FAQ:** Часто задаваемые вопросы.
* **Contact:** Форма обратной связи для потенциальных клиентов.

## 🚀 Функциональность

* **Адаптивная верстка:** Корректное отображение на десктопах и мобильных устройствах.
* **Информационные блоки:** Структурированное описание преимуществ системы.
* **Интеграция с Google Sheets:** Форма на странице "Контакты" подключена к Google Таблицам. Заявки автоматически сохраняются в таблицу без использования серверного бэкенда.Здесь вы можете увидеть саму таблицу, куда данные отправляются:
